# C+ Syntax & Data Types

Learning the basic syntax of C+ is the first step to writing your first program.

## Identifiers & Keywords
In C+, keywords like `LET`, `PRINT`, `IF`, and `FOR` must be written in **UPPERCASE**. This helps distinguish them from variables and other names you create.

## Variables
Use the `LET` keyword to define a variable. Variables store values for later use.
```basic
LET age = 25
LET name = "C+ Language"
```

## Data Types
C+ supports two primary data types:

### 1. Numbers
All numbers in C+ are treated as 64-bit floating-point numbers. They can be whole numbers or decimals.
```basic
LET count = 10
LET price = 19.99
```

### 2. Strings
Strings are sequences of characters enclosed in double quotes.
```basic
PRINT "Hello, World!"
```

## Comments
Currently, the C+ interpreter focuses on core execution. We recommend using descriptive variable names to keep your code readable.
