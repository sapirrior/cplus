# Welcome to C+

C+ is a modern, production-grade BASIC-style interpreter written in C++17. It's designed to be fast, simple, and easy to learn.

## Design Philosophy
C+ aims to bridge the gap between classic BASIC simplicity and modern programming practices. We've removed line numbers (e.g., `10 PRINT "Hello"`) in favor of a clean, structured syntax that's easier to read and maintain.

## How it Works
1. **Lexer**: Breaks your code into tokens (the basic building blocks like keywords, numbers, and strings).
2. **Parser**: Organizes those tokens into an Abstract Syntax Tree (AST) to understand the structure of your program.
3. **Interpreter**: Executes the AST, bringing your code to life.
